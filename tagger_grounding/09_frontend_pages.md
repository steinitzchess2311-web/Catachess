# Stage 09 - 前端 Players 页面

目标：实现产品级 UI。

## 一、路由
- /players → PlayersIndex
- /players/:id → PlayerDetail

## 二、PlayersIndex
- 列表展示 player profiles
- 新建按钮
- 显示已分析位置数与更新时间

## 三、PlayerDetail
- 棋手名 + 别名
- 状态（Idle / Processing / NeedsConfirmation / Error / CompletedWithErrors）
- 上传区（拖拽 + 按钮）
- 进度条 + 日志
- 统计区（Total/White/Black）
- Export 按钮

## Checklist
- [ ] 路由与导航确认
- [ ] 上传区 UI 完整
- [ ] 统计区 Tab 完整
- [ ] 状态展示完整
